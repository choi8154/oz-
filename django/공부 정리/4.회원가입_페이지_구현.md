# 회원가입 페이지 구현

### 1) 회원가입 app 및 라우터 생성

- 다른 기능의 라우터를 넣을 앱 모듈을 생성 : python manage.py startapp member

- views에 엔드포인트 함수 생성
```python
from django.shortcuts import render
from django.contrib.auth.forms import UserCreationForm

def sign_up(request):
    form = UserCreationForm()

    context = {
        'form':form
    }

    return render(request, "signup.html", context)
```

- settings에 app 모듈 등록

- urls에 해당 라우터 등록 : path('signup/', <함수>, name = "<이름>")


### 2) 회원가입 페이지 구현

- 템플릿 registration에 html 생성
```html
<body>
    <form method = "POST">
        {% csrf_token %}
        {{ form.as_p }}
        <button>회원가입</button>
    </form>
</body>
```

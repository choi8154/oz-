# DRF초기 설정 순서

## 1. 프로젝트 생성 및 가상환경 설정
1. pyenv virtualenv <버전> <환경이름> : python 버전 가상환경
2. pyenv local <환경이름> : 가상환경 설정
3. poetry init : 포에트리 가상환경 초기화
4. poetry install : 포에트리 설치
5. poetry add django djangorestframework : 장고 DRF 설치

## 2. Django 프로젝트 시작 및 setting 설정
1. django-admin startproject config . : 프로젝트 폴더 생성, django프로젝트 시작
2. django settings 설정 
3. python manage.py startapp <폴더이름> : 프로젝트 모듈 생성

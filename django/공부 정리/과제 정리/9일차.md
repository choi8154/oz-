## 1. Django 프로젝트 세팅

### 1) 가상환경 구축하는 방법 (with Poetry)
- pyenv virtualenv <버전> <이름> : 파이썬 가상환경
- pyenv local <이름> : 현재 프로젝트를 파이썬 가상환경 폴더에 지정
- poetry init : poetry 초기화
- poetry install : poetry 설치


### 2) Django 설치 및 프로젝트 구성, 앱 세팅하는 방법
- poetry add django djangorestframework : 장고, DRF 설치
- django-admin startproject <폴더이름> . : 장고 프로젝트 시작
- python manage.py startapp <폴더이름> : app시작
- settings에 INSTALLED_APPS에 app 추가

```py
from django.db import models

class Data(models.Model):
    title = models.CharField(max_length=50)
    content = models.TextField(blank=True)

```
### 2) 각 필드별 특징 정리하기
- `CharField` : 짧은 문자
- `TextField()` : 긴 문자
- `IntegerField` : 숫자
- `BooleanField` : True,False
- `DateTimeField` : 날짜
- `OneToOneField` : 1대1 관계
- `ManyToManyField` : 다대다 관계
- `ForeignKey` : 1대다 관계


### 3) Migration에 대해서 정리하기
- 마이그레이션은 테이블 버전 관리 시스템으로 모델에서 발생한 스키마 변화를 migrations폴더에 버전으로 기록하고 적용하는 DB 스키마 버전 관리 시스템임.
- `python manage.py makemigrations` : 마이그레이션 생성(버전 생성)
- `python manage.py migrate` : 마이그레이션 최신 기록을 DB테이블에 적용

## 4. FBV(Function Base View)

### 1) FBV란?
- Function Based View로 함수별로 엔드포인트 기능을 관리하는 방법

### 2) 간단한 View 메소드 작성법
- render, redirect
```py
from django.shortcuts import render
from .forms import BlogCreateForm

def blog_create(request):
    form = BlogCreateForm(data = request.data)
    if form.is_valid():
        blog = form.save(commit=False)
        blog.save()
        return redirect('blog_list')
    context = {'form':form}
    return render(request, '어쩌고.html', context)
```

## 5. Urls

### 1) URL 엔드포인트란?
- 해당 라우트로 요청이 들어오면 함수를 실행시키는 엔드포인트
### 2) urlspatterns
- Django에서 URL 라우트를 등록해두는 리스트
```py
from django.urls import path

urlpatterns = [
    path('post/', blog_create, name='blog_create'),
]
```
### 3) URL Include 개념 및 방법
- 앱에서 관리하는 urls.py의 라우터를 모두 등록함.
- fastapi의 APIRouter랑 같은 역할임
```py
urlpatterns = [
    path('', include('blog.urls'))
]
```
